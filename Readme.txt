This repository contains scripts used in the analyses of "Coupled phylogenetic and functional enrichment in the tomato rhizosphere microbiome" by Talavera-Marcos S, Gallego R, Rastrojo A, and Aguirre de Cárcer D.

	Use remove_341F_805R.py

2-Initial sample processing 
	dada2multithreads.R
	
3-Process sequences and analyses
